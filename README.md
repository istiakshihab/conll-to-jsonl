## Convert CONLL formatted NER file to JSONL formatted NER file

With this py file you can easily convert CONLL formatted NER Dataset file to JSONL formatted NER Dataset file (example: https://github.com/csebuetnlp/banglabert/blob/master/token_classification/sample_inputs/validation.jsonl).

```
python conll-to-jsonl.py input.conll output.jsonl
```
